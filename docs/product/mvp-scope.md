# MVP Scope

## Must have

- Multi-tenant organization setup
- FAQ/knowledge base search
- Voice webhook
- Bangla/English/Banglish language routing
- Sylhet-friendly term normalization
- Unknown question logging
- Lead capture
- Human handoff
- Admin dashboard

## Not in MVP

- Fine-tuned dialect ASR
- Full billing automation
- Enterprise SSO
- Advanced outbound campaigns
